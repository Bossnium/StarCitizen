Star Citizen Automation

Prerequisites

  • Python (version 3.6 or above) - Python Downloads

Dependencies

The following dependencies are required for running the script:

  • pywin32: pip install pywin32
  • pywinauto: pip install pywinauto
  • keyboard: pip install keyboard

Installation

  1. Install Python from the official Python website: Python Downloads
  2. Install the required dependencies using pip. Open a command prompt or terminal and run the following commands: 
            pip install pywin32
        pip install pywinauto
        pip install keyboard

Usage

  1. Open a command prompt or terminal.
  2. Navigate to the directory where the script is located.
  3. Run the script using the following command: 
    python star_citizen_automation.py
  4. The script will start running and periodically check the Star Citizen window. It will perform certain actions based on the game's status and your manual key combinations.
  5. Press LCtrl + RAlt to trigger the manual action of sending /showlocation and extracting the coordinates.
  6. The script will log the actions and their statuses in the console and also write them to a log file (log.txt).
